// 01 — Motivation

Pourquoi le développement ?

Ce qui me pousse à construire, et ce que je cherche à bâtir.

J'ai découvert le développement web comme on découvre quelque chose qui fait sens : la logique, la précision, le fait que chaque ligne de code produit quelque chose de concret et visible. Ce rapport direct entre ce qu'on écrit et ce qu'on crée — c'est ce qui me retient.

Ce qui m'attire dans le développement fullstack, c'est justement cette vision d'ensemble : comprendre comment le front dialogue avec le back, comment une API est conçue pour être robuste, comment une base de données est modélisée pour tenir dans le temps. Je n'aime pas faire les choses à moitié — je veux comprendre tout le chemin, de la requête à l'interface.

Je cherche une alternance où je pourrai contribuer à de vrais projets, m'améliorer sur les bonnes pratiques — tests, code review, documentation — et grandir aux côtés de développeurs qui ont l'envie de transmettre.
